### Installation

```
git clone https://github.com/SpookyGang/Stella.git
cd Stella
pip3 install -U requirements.txt
```
- Rename `Stella/sample_config.py` to `Stella/config.py`
- Fill all the variables
```
python3 -m Stella
```
